# CV/Resume Upload Directory

This directory stores uploaded candidate resumes (PDF files).

## File Naming Convention
`{original-name}-{timestamp}-{random}.pdf`

Example: `john-doe-resume-1699123456789-987654321.pdf`

## File Size Limit
Maximum 5MB per file

## Allowed File Types
- PDF only

## Security Notes
- Files are validated before upload
- Only authenticated RECRUITER/ADMIN can upload
- File paths are stored in `candidate_resumes` table
